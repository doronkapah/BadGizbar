Firebase Setup Required

Open app.js and fill in your Firebase project config. Follow these steps:

  1. Go to console.firebase.google.com
  2. Create a project → click Add web app
  3. In the left sidebar, go to Build → Realtime Database
  4. Click Create Database → choose a region → Start in test mode
  5. Go to Project settings (gear icon) → scroll down to Your apps
  6. Copy the firebaseConfig object and paste the values into app.js
